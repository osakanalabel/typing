# タイピストへの道

小学校高学年向け、日本語タイピング練習ゲーム。漢字まじりの文章を見て、その読みをひらがなで打ち込む。

**公開URL: <https://osakanalabel.github.io/typing/>**

## 特徴

- 漢字まじりの文章を読んで、ひらがなで答えるタイピングゲーム
- 1ターン10問、終わるとリザルトと獲得経験値を表示

## あそびかた

1. キーボードがあるパソコンで上記URLを開く
2. **はじめる** を押して10問チャレンジ
3. 出題された文章を見て、その読みをひらがなで入力 → Enter
4. **Esc** で空欄スキップ(不正解扱い)
5. 編集距離 3 以下なら **おしい!** で半分の経験値がもらえる

## 動作環境

- PC のキーボード必須(スマホ・タブレット非対応)
- モダンブラウザ(Chrome / Edge / Firefox / Safari)
- 古い Chromebook でも動くよう、画像・SVG なしの軽量実装
